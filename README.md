- To install the python requirements:
```
pip install -r requirements.txt 
```

- Running CFR, XDO and XODO on Kuhn Poker
```
cd train
python3 cfr.py
python3 xdo.py
python3 oxdo.py
```
- The exploitability results are in the plot file. Then to get a exploitability figure, just run
```
cd ../plot
python3 plot.py
```
